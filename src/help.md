# Markdown Cheat Sheet

## Headings
```
# H1
## H2
### H3
```

## Emphasis
```
**bold**  or  __bold__
*italic*  or  _italic_
~~strikethrough~~
```

## Lists
```
- Unordered item
- Another item
  - Nested item

1. Ordered item
2. Second item
```

## Links & Images
```
[Link text](https://example.com)
![Alt text](image.png)
```

## Code
```
`inline code`

    indented code block (4 spaces)
```

Fenced block:
````
```rust
fn main() {}
```
````

## Blockquote
```
> This is a blockquote
```

## Horizontal Rule
```
---
```

## Table
```
| Column A | Column B |
|----------|----------|
| cell 1   | cell 2   |
```
